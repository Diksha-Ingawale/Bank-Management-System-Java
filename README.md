# 💳 Bank Management System (Java)

A simple **console-based Bank Management System** built using **Core Java**.  
This project allows users to create accounts, deposit or withdraw money, and view transaction history — all data is stored locally in a text file.

---

## 🚀 Features
- 🧾 Create New Bank Accounts  
- 🔐 Login with Account Number and Password  
- 💰 Deposit and Withdraw Money  
- 📜 View Transaction History  
- 💾 File-based Data Storage (Persistent)

---

## 🛠️ Technologies Used
- Java (Core)
- File Handling (I/O)
- OOPs Concepts (Classes, Objects, Methods)

---

## 🖥️ How to Run the Program

### Step 1: Compile the code
javac BankManagementSystem.java

### Step 2: Run the program
java BankManagementSystem
